# Temperature Converter

A Python command-line application that converts temperatures between Fahrenheit and Celsius and categorizes the temperature.

## Features
- Fahrenheit to Celsius conversion
- Celsius to Fahrenheit conversion
- Temperature category detection (Freezing, Cool, Warm)
- Error handling for invalid input
- Menu-driven interface that loops until the user exits

## Technologies Used
- Python

- Conditional statements
- Loops
